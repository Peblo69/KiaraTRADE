Now you can run the application in different ways:

For development (with auto-reload):
bash

npm run dev          # Run main application
npm run dev:monitor  # Run monitor
npm run dev:tracker  # Run tracker

For production:
bash

npm run prod          # Build and run main application
npm run prod:monitor  # Build and run monitor
npm run prod:tracker  # Build and run tracker

For testing:
bash

npm test

To clean and rebuild:
bash

npm run clean    # Remove dist folder
npm run build    # Rebuild TypeScript
npm run rebuild  # Rebuild native modules if needed
